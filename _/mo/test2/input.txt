오퍼레이션 타입 0, 1 이 있다.
input<int>[] = [operationType, y, x]
오퍼레이션 0) 은 주어진 input[] = [0, y, x] 에서 y X x 의 사각형을 그리고 배열에 저장
오퍼레이션 1) 은 주어진 input[] = [1, y, x] 에서 y X x 사각형이 그동안 저장된 사각형들의 배열에서
모두 fit 한지, 내부에 들어가는지 확인해서 결과배열에 bool 저장
최종적으로 결과값만 저장된 배열을 리턴
# input
[0, 2, 3]
[0, 4, 2]
[0, 7, 9]
[1, 2, 4]
[1, 2, 2]

# result
[false, true]
